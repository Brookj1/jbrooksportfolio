Digital Bibliophile

Project link

Digital Bibliophile

Group project created with three of my bootcamp classmates. Beta version of app (MVP) that allows users to search for digital books and check their availability using the Digital Public Library of America and Open Library APIs. Future iterations would include integrations with OverDrive and Google Maps to allow users to search their local libraries for books and link to their sites for quick and painless digital checkout. For this project we all worked on every aspect of the app, including HTML, CSS, Materialize, JavaScript/jQuery, APIs, Ajax, and JSON. We used GitHub for collaboration and version control, and Trello and Slack for project management.
